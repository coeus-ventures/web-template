# Run

Instructions: $ARGUMENTS

You will complete the issue. You will first update the issue file only with the plan command, once it's updated you will run the execute command to actually implement it.

1. Use the command /plan with the issue file as argument
2. Use the command /execute with the issue file to implement the issue
